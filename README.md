🎨 Visual Branding & A/B Testing ROI Analytics

📌 Project Overview
This project bridges the gap between creative design and performance marketing using data analytics. By evaluating a simulated dataset of 3,000 ad campaign records spanning one year, this interactive dashboard measures the financial return (ROAS) of 4 distinct visual branding styles ('Modern Minimalist', 'Bohemian Vibe', 'Aggressive Bold', 'Corporate Trust').

🚀 Key Business Insights
The "Clickbait" Trap: Data reveals that 'Aggressive Bold' designs generate the highest impressions and clicks, but fail drastically in conversions, leading to a poor Return on Ad Spend (ROAS).
Winning Aesthetic: The 'Bohemian Vibe' visual style emerged as the definitive winner. Despite lower initial impressions, it maintained an exceptionally high conversion rate, yielding the highest overall ROAS.
Cost Efficiency: Visualized the relationship between CPC (Cost Per Click) and CTR (Click-Through Rate) using a scatter plot, providing actionable insights for Media Buyers and Art Directors.
Creative Agency UI: Developed a premium, glassmorphism-inspired HTML dashboard with neon accents to reflect a modern creative agency environment.

🛠️ Tech Stack
Language: Python 3.x
Data Manipulation: Pandas, NumPy
Data Visualization: Plotly (Scatter Bubble Chart, Horizontal Bar, Line Trends)
UI/UX: Standalone HTML/CSS Grid Dashboard (Neon/Dark Mode)
